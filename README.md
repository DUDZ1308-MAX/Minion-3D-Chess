# 3D Chess - Portable Edition

A fully playable 3D chess game that runs anywhere - no installation required!

## 🚀 Quick Start

**Double-click `Launch_3D_Chess.vbs`** — that's it! The game will:
1. Auto-detect Node.js (uses high-performance server if available)
2. Fall back to built-in PowerShell (works on any Windows 7+)
3. Start the server and open your browser to `http://localhost:8080/`

## 🎮 Alternative Launchers

| File | Description |
|------|-------------|
| `Launch_3D_Chess.vbs` | **Best** - Silent, no console window, auto-opens browser |
| `LAUNCH.bat` | Shows console output, good for debugging |
| `run.bat` | Original launcher |
| `run.sh` | Linux/macOS launcher |
| `server.ps1` | Direct PowerShell server |
| `node server.js` | Node.js server (fastest) |

## ✨ Features

- **Full chess rules**: Castling, en passant, promotion, check/checkmate/stalemate
- **3D animated pieces**: Smooth move animations + capture explosions
- **8 beautiful themes**: Classic, Emerald, Ocean, Sunset, Forest, Royal, Midnight, Candy
- **Move history**: Timestamped log with algebraic notation
- **Game clock**: Elapsed time tracker
- **Keyboard + mouse controls**
